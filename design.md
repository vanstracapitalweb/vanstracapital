# Vanstra Capital - Design Style Guide

## Design Philosophy

### Visual Language
Vanstra Capital embodies **elite financial sophistication** through a carefully crafted visual system that balances **trust, security, and modern innovation**. The design language draws inspiration from premium financial institutions while establishing a unique identity that feels both timeless and forward-thinking.

### Core Principles
- **Trust & Security**: Every visual element reinforces reliability and financial stability
- **Premium Quality**: Sophisticated aesthetics that convey exclusivity and high-value service
- **Modern Innovation**: Clean, contemporary design that signals technological advancement
- **Accessibility**: Inclusive design that serves diverse user needs while maintaining elegance

## Color Palette

### Primary Colors
- **Midnight Navy (#041225)**: Primary brand color, conveys trust and stability
- **Deep Slate (#0B2A3F)**: Secondary dark tone, used for depth and hierarchy
- **Gold Accent (#C89A3A)**: Premium accent color, represents prosperity and success
- **Soft White (#F7F8FA)**: Primary background and text color
- **Gray (#8B96A3)**: Neutral tone for secondary text and subtle elements

### Color Usage Guidelines
- **Backgrounds**: Midnight Navy for hero sections, Soft White for content areas
- **Text**: Soft White on dark backgrounds, Midnight Navy on light backgrounds
- **Accents**: Gold used sparingly for CTAs, highlights, and premium indicators
- **Gradients**: Subtle navy-to-slate gradients for depth, never gold gradients

## Typography

### Primary Typeface: Modern Serif (Headings)
- **Font Family**: Playfair Display, Cormorant Garamond, or similar
- **Usage**: Main headings, hero text, section titles
- **Characteristics**: Elegant, authoritative, premium feel
- **Weights**: Regular (400), Medium (500), Bold (700)

### Secondary Typeface: Clean Sans-Serif (Body)
- **Font Family**: Inter, Source Sans Pro, or similar
- **Usage**: Body text, navigation, buttons, forms
- **Characteristics**: Highly readable, modern, neutral
- **Weights**: Regular (400), Medium (500), Semibold (600)

### Typography Scale
- **H1**: 3.5rem (56px) - Hero headings
- **H2**: 2.5rem (40px) - Section headings
- **H3**: 1.875rem (30px) - Subsection headings
- **H4**: 1.5rem (24px) - Card titles
- **Body**: 1rem (16px) - Standard text
- **Small**: 0.875rem (14px) - Captions, metadata

## Visual Effects & Styling

### Used Libraries & Effects
- **Anime.js**: Smooth micro-interactions and button animations
- **ECharts.js**: Financial data visualization with custom dark theme
- **Pixi.js**: Particle effects for premium background elements
- **Splitting.js**: Text reveal animations for headings
- **Splide.js**: Elegant carousels for testimonials and features

### Animation Principles
- **Subtle Motion**: All animations serve a functional purpose
- **Easing**: Custom cubic-bezier curves for premium feel
- **Duration**: 200-400ms for micro-interactions, 600-800ms for reveals
- **Stagger**: Sequential reveals for grouped elements

### Header Effects
- **Gradient Background**: Subtle navy-to-slate gradient with animated particles
- **Text Effects**: Gold accent underlines that draw in on scroll
- **Interactive Elements**: Hover states with gentle scale and shadow changes

### Card Design
- **Background**: Soft white with subtle transparency on dark sections
- **Borders**: Minimal 1px borders in slate with gold hover states
- **Shadows**: Soft, layered shadows for depth without heaviness
- **Corners**: 12px border radius for modern, approachable feel

### Interactive Elements
- **Buttons**: Gold accent backgrounds with white text, subtle hover lift
- **Forms**: Clean inputs with gold focus states and floating labels
- **Navigation**: Minimal underline indicators with smooth transitions
- **Hover States**: Gentle scale (1.02x) with shadow expansion

## Layout & Grid System

### Grid Structure
- **12-Column Grid**: Flexible layout system for all screen sizes
- **Breakpoints**: Mobile (320px), Tablet (768px), Desktop (1024px), Large (1440px)
- **Spacing**: 8px base unit system (8, 16, 24, 32, 48, 64px)
- **Margins**: 16px mobile, 32px desktop

### Content Hierarchy
- **Hero Sections**: Full-width with centered content, max-width 1200px
- **Content Blocks**: Alternating layouts with generous white space
- **Cards**: Equal-height grid layouts with consistent spacing
- **Forms**: Single-column mobile, multi-column desktop

## Component Styling

### Navigation
- **Position**: Sticky header with backdrop blur effect
- **Background**: Semi-transparent midnight navy
- **Typography**: Clean sans-serif, medium weight
- **Hover States**: Gold underline animation

### Buttons
- **Primary**: Gold background, white text, subtle shadow
- **Secondary**: Transparent background, gold border, gold text
- **Hover**: Gentle lift effect with expanded shadow
- **Disabled**: Reduced opacity with no interaction

### Cards
- **Background**: Soft white with subtle border
- **Padding**: 32px all sides for generous breathing room
- **Hover**: Gentle lift with enhanced shadow
- **Content**: Clear hierarchy with consistent spacing

### Forms
- **Inputs**: Clean borders with gold focus states
- **Labels**: Floating animation on focus
- **Validation**: Green success, red error with icons
- **Spacing**: 24px between form groups

## Accessibility & Contrast

### Color Contrast Ratios
- **Primary Text**: 4.5:1 minimum contrast ratio
- **Secondary Text**: 3:1 minimum contrast ratio
- **Interactive Elements**: Clear visual focus indicators
- **Error States**: High contrast red with sufficient text contrast

### Motion & Animation
- **Reduced Motion**: Respect user preferences
- **Focus States**: Clear, high-contrast focus rings
- **Loading States**: Accessible loading indicators
- **Error Handling**: Clear, descriptive error messages

## Brand Applications

### Logo Usage
- **Primary**: Gold logo on dark backgrounds
- **Secondary**: Navy logo on light backgrounds
- **Minimum Size**: 120px width for digital applications
- **Clear Space**: Logo height equivalent spacing around all sides

### Photography Style
- **Tone**: Professional, trustworthy, aspirational
- **Lighting**: Natural, soft lighting with subtle gold tones
- **Composition**: Clean, uncluttered with generous negative space
- **Color Grading**: Warm, premium feel complementary to brand colors
